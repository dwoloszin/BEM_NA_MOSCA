## 💡 O Problema

No Brasil, os preços dos supermercados mudam quase todo dia. A gente perde muito tempo — e dinheiro — andando de mercado em mercado ou comparando preços no celular.

Muitas famílias acabam gastando **até 30% a mais** do que precisariam, simplesmente por não saber onde está mais barato naquele momento.

---

## ✅ A Solução

O **Bem na Mosca** é um aplicativo **gratuito** que:

- 🔄 **Reúne os preços** de vários supermercados e atualiza todo dia
- 📋 **Você cadastra sua lista de compras mensal** uma única vez
- 🏪 **Te mostra qual é o melhor mercado** para fazer suas compras naquele dia (com a maior economia possível)
- 📅 **Indica os melhores dias da semana** para comprar cada item
- 📊 **Mostra os períodos de melhores promoções** de cada mercado, com base em histórico real

### Supermercados mapeados (em expansão):

| | | |
|---|---|---|
| Atacadão | Extra | Higas |
| Nagumo | Sonda | Oba Hortifruti |
| Pão de Açúcar | Rossi | Barbosa |
| Carrefour | Swift | ... e mais |

> 📍 Dados **regionalizados por loja** — a comparação é feita com os mercados mais perto de você.

## 🚀 Funcionalidade já disponível: Fast List

**Acesse agora:** [dwoloszin.github.io/PricePilot/#/FastList](https://dwoloszin.github.io/PricePilot/#/FastList)

Sabe aquela sensação ruim de chegar na fila do caixa sem ideia de quanto vai dar o total? O **Fast List** resolve isso!

Com apenas 4 passos:

1. 📷 Escaneie o código de barras do produto
2. 💰 Informe o valor
3. 🔢 Informe a quantidade
4. ➕ Adicione na lista

O app soma tudo automaticamente. Antes mesmo de chegar no caixa, você já sabe o valor exato da sua compra — e consegue conferir se a promoção da gôndola bate com o que está no sistema do caixa.

**Simples. Rápido. Prático. Zero ansiedade.**
